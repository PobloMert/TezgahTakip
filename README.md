# Tezgah Takip Programı (Qt Sürümü)

## Kurulum

1. Python 3.8+ yüklü olmalı

2. Gerekli kütüphaneleri yükleyin:
```bash
pip install -r requirements.txt
```

## Çalıştırma

### Windows
- `start.bat` dosyasını çift tıklayın
- Veya komut satırından:
```bash
python main.py
```

### Linux/Mac
```bash
python3 main.py
```

## Özellikler
- Tezgah bakım takibi
- Pil değişim kayıtları
- Detaylı raporlama
- Otomatik güncelleme

## Geliştirme
